# LISOC

A management system for Long Island School of Chinese (长岛中文学校)

#### Built With

**Next.js 15**, **TypeScript**, **Tailwind CSS**, **Drizzle ORM (PostgreSQL)**

_Libraries:_ Next-Auth, React Hook Form, Zod, PayPal SDK

## Getting Started

1.  **Clone the repository:**

    ```bash
    git clone [https://github.com/ryw27/lisoc.git](https://github.com/ryw27/lisoc.git)
    ```

2.  **Install dependencies:**

    ```bash
    npm install
    ```

3.  **Setup Environment:**
    Copy `.env.example` to `.env.local` and add your credentials.

    ```bash
    cp .env.example .env.local
    ```

4.  **Run the development server:**
    ```bash
    npm run dev
    ```
